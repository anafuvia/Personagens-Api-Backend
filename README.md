## 🎯 Tema da API

Personagens de desenhos animados.



## 📖 Descrição do projeto

Neste projeto desenvolvemos uma API simples que fornece informações sobre personagens de desenhos animados.

Além disso, criamos um **front-end** responsável por consumir essa API e exibir os dados de forma visual e organizada na tela.

O objetivo do trabalho foi praticar:

* Criação de API com Node.js
* Comunicação entre backend e frontend
* Manipulação de dados em formato JSON

---

## 🛠️ Tecnologias utilizadas

* Node.js
* Express
* JavaScript
* HTML
* CSS
* JSON

## 🚀 Como executar o Backend

1. Abrir o terminal
2. Entrar na pasta do backend:

```
cd backend
```

3. Instalar as dependências:

```
npm install

4. Iniciar o servidor:
node server.js

5. A API estará rodando em:

http://localhost:3000

## 💻 Como executar o Frontend

1. Entrar na pasta do frontend:
cd frontend

2. Abrir o arquivo:
index.html


3. O navegador irá carregar a aplicação e consumir a API automaticamente.
